# Development of a Text and Audio Corpus for the Task of Natural Language Understanding (NLU) in Russian

Данный репозиторий содержит код и материалы по project proposal.

**Автор**: Заряна Дамашова

## Структура репозитория
### ↬ Предобработка данных
**`clean_up.ipynb`**  
Очистка и нормализация оригинального английского датасета XSID ( Van der Goot et al., 2021). 

**`translation.py`**  
Перевод очищенного тренировочного датасета на русский язык через API.

**`adaptation.ipynb`**  
Адаптация переведенного тренировочного датасета.

### ↬ Полученные данные

**`en.train.reference.conll`**  
Очищенный от дубликатов + с добаленными IDs оригинальный английский датасет.

**`en.train.unique_ids.conll`**  
Файл с удаленными слотами + интентами внутри токенов для перевода на русский

**`ru.train.conll`**  
Переведенный русский датасет

**`ru.train_adapt.conll`**
Адаптированная версия переведенного датасета
